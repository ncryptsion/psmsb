# PSMSB
Philippines Phone Number SMS Bomber.

# ⚙️ Installations
## Github
```
git clone https://github.com/ncryptsion/psmsb.git
```

## NpmJS
```
npm install
```

## PNPM
```
pnpm install
```

# 🚀 Usage
```
node index.js <phoneNumber> <amount>
```

| Argument | Description | Example |
| --- | --- | --- |
| phoneNumber | The target phone number. | `node index.js 09123456789 <amount>` |
| amount | How many times to spam the phone number. | `node index.js 09123456789 10` |
